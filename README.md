# CryptoCurrency-Portfolio-Manager
Desktop application for managing cryptocurrency portfolios, using the CoinMarketCap API for live price tracking and SQLite3 to store user-selected coins.

🚀 Features

Real-time cryptocurrency price tracking
Portfolio value calculation
Profit/Loss per coin calculation
Total portfolio profit/loss summary
SQLite database integration
Clean tabular UI using Tkinter Grid system
Update button to refresh live market data
Basic API error handling
